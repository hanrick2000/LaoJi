# Two Pointers

## Forwards two pointers

### On arrays

master pointer i - for loop
\
slave pointer j - while loop

Algorithm:
  1. move j (while we should include element j)
  2. save result (arr[i:j])
  3. move i (in for loop)

Time O(n)
