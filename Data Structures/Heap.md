# Heap / Priority Queue

## Overview 

Get min/max in time O(1)
Update in time O(logn)
Remove min/max in time O(logn)
Building (bottom up) in time O(n)



__Examples:__

[295. Find Median from Data Stream](https://leetcode.com/problems/find-median-from-data-stream/)
